# Kana Cards

Offline-first flashcard web app for learning hiragana & katakana.
Front shows the English sound (romaji) + a canvas to draw the character; flip
reveals the character with animated stroke order and Good/Okay/Bad grading.
Scheduling uses **FSRS**. Multi-profile, progress stored on-device (localStorage),
works with no network. Built for my kids.

**Stroke-order data:** [KanjiVG](https://kanjivg.tagaini.net) © Ulrich Apel,
licensed CC BY-SA 3.0. This project reuses that data under the same license.
